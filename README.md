# Simple PHP Blog  

## Requirements  
- **Laragon** (or similar stack like XAMPP/WAMP)  
- **PHP 7.4+**  
- **MySQL/MariaDB**  

## Setup Instructions  

1. **Clone the project**  
   ```bash
   git clone https://github.com/mohamad-21/simple-php-blog.git
   ```

2. **Configure database connection**  
   - Open `config.php`.  
   - Update the host, username, password and database name.  

4. **Run the project**  
   - Start Apache and MySQL.  
   - Open your browser and go to:  
     ```
     http://localhost/simple-php-blog
     ```
